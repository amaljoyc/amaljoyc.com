## Challenge

You are given an m x n 2D image matrix where each integer represents a pixel. Flip it in-place along its vertical axis.

**Example 1:**

Input image :  

              1 0
              1 0

Modified to :

              0 1
              0 1

**Example 2:**

Input image :  

              1 2 3
              4 5 6
              7 8 9

Modified to :

              3 2 1
              6 5 4
              9 8 7
