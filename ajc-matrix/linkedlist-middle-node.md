## Challenge

Given a Singly-Linked List, write a method - findMiddleNode that finds and returns the middle node of the list in a single pass.

**Examples:**
1 ==> 1
1->2 ==> 1
1->2->3->4 ==> 2
1->2->3->4->5 ==> 3
